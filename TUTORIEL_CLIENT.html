Tutoriel Espace Client

Carte Limap — Guide rapide pour consulter les producteurs et envoyer vos demandes de mise à jour.

1. Accès à l'espace client

Ouvrez la page suivante dans votre navigateur :

https://limap.fr/Carte/client/index.html

  1. Saisissez votre identifiant et votre mot de passe.
    Ces informations vous sont envoyées par Limap.
  2. Cliquez sur Se connecter.
    Vous arrivez sur votre espace avec la liste des producteurs liés à votre structure.
Astuce : si vous ne voyez pas vos dernières modifications, rechargez la page avec Ctrl + F5.

2. Que pouvez-vous faire dans l'espace client

Profil client

Mettre à jour les informations de votre structure (coordonnées, site web, etc.).

Fiches producteurs

Renseigner vos informations de relation client pour un producteur (notes, statut, catégories utiles pour votre suivi).

Demande de modification globale

Proposer une correction sur une fiche producteur (adresse, contact, activités, etc.).

Demande de création

Proposer un nouveau producteur à ajouter à la carte.

Rattachement fournisseur existant

Demander à ce qu'un producteur déjà présent dans la base soit rattaché à votre structure.

3. Comprendre la fiche globale du producteur

Chaque producteur possède une fiche globale unique dans la plateforme.

  • Cette fiche contient les informations communes à tous les clients : nom, adresse, téléphone, email, site web, activités, labels.
  • L'objectif est d'éviter les doublons et d'avoir une base fiable partagée.
  • Les modifications de cette fiche passent par une validation admin pour garantir la qualité des données.
En plus de la fiche globale, votre espace client contient vos propres informations de relation (notes, statut, suivi), visibles pour votre structure uniquement.

4. Modifier une fiche producteur (votre espace client)

  1. Ouvrez le producteur concerné.
    Utilisez la recherche ou la liste.
  2. Modifiez les champs de votre profil client.
    Exemple : statut de relation, notes internes, catégories de suivi.
  3. Enregistrez.
    La mise à jour est immédiate dans votre espace client.
Important : certaines modifications touchant la fiche globale du producteur passent par une validation admin.

5. Envoyer une demande de modification globale

  1. Choisissez le producteur.
  2. Sélectionnez le champ à corriger.
    Exemple : téléphone, email, adresse, site web, activités, labels.
  3. Saisissez la nouvelle valeur et, si possible, une justification.
  4. Envoyez la demande.
    Elle apparaît en statut « En attente ».

Une fois traitée par l'administrateur, le statut passe en « Approuvée » ou « Refusée ».

6. Vérifier qu'un producteur n'existe pas déjà

  1. Depuis l'espace client, vérifiez d'abord votre périmètre.
    Testez plusieurs mots-clés : nom exact, nom partiel, ville.
  2. Puis vérifiez sur la carte publique Limap :
    • Carte interactive : https://limap.fr/map/
    • Page producteurs : https://limap.fr/producteurs/
  3. Vérifiez les variantes d'écriture.
    Exemple : GAEC, EARL, accents, tirets, abréviations.
Règle importante : si le producteur existe déjà, ne pas créer une nouvelle fiche.

7. Si le producteur existe déjà mais n'est pas lié à votre client

Vous pouvez envoyer une demande de rattachement directement depuis votre espace client, sans contacter Limap par email.

  1. Dans votre espace client, allez sur l'onglet « Nouveau fournisseur ».
  2. Faites défiler jusqu'à la section « Rattacher un fournisseur déjà existant ».
  3. Tapez le nom du producteur dans le champ de recherche (au moins 2 caractères).
    Les résultats affichent les producteurs de la base qui ne sont pas encore liés à votre structure.
  4. Identifiez le bon producteur et cliquez « Demander le rattachement ».
    Vous pouvez ajouter une note pour donner du contexte à l'administrateur.
  5. Attendez la validation admin.
    Le producteur apparaîtra ensuite dans votre liste.
Votre demande est visible dans l'historique juste en dessous, avec son statut (En attente, Approuvée, Refusée).

8. Proposer un nouveau producteur

Si le producteur n'existe pas encore dans la base (après vérification, cf. section 6), vous pouvez proposer sa création.

  1. Allez sur l'onglet « Nouveau fournisseur ».
  2. Renseignez les informations principales.
    Nom, localisation, contact, activités, type, site web.
  3. Envoyez la demande.
    L'administrateur vérifie puis valide ou refuse. En cas de validation, le producteur sera créé et rattaché à votre client.
Vous avez une grande base de données existante ?
Si vous avez déjà une liste importante de producteurs (dans un fichier Excel par exemple), il est beaucoup plus rapide de nous envoyer directement ce fichier — nous pouvons importer tous vos producteurs en une seule opération, sans que vous ayez à tout saisir un par un. Contactez l'équipe Limap pour en discuter.

9. Mot de passe oublié

  1. Depuis la page de connexion, cliquez sur « Mot de passe oublié ».
  2. Saisissez votre email utilisateur.
  3. Consultez votre boîte mail et suivez le lien de réinitialisation.
  4. Définissez un nouveau mot de passe.
Si vous ne recevez pas le message, vérifiez les dossiers Spam et Promotions.

10. Bonnes pratiques

  • Vérifier les informations avant envoi (email, téléphone, ville, code postal).
  • Ajouter un commentaire clair pour accélérer la validation admin.
  • Éviter les doublons : chercher un producteur avant de proposer une création.
  • Ne pas partager votre mot de passe.

11. Support

En cas de problème de connexion ou de doute sur une demande, contactez l'équipe Limap.

Incluez si possible : nom du compte, page concernée, message d'erreur, et capture écran.
